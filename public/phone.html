Hard refresh

Clears everything stored by Pip in this browser and reloads. Use this when the page is stuck in a broken state that normal reload doesn't fix.

This will erase:
  • App cache and service worker (re-downloaded on reload)
  • Phone identity key (you'll need to re-pair to your Mac)
  • Trust store (which Macs this phone has trusted)

If you installed Pip to your Home Screen, the install survives — only the data inside it is cleared.

Tap the Share button in Safari, then Add to Home Screen.
