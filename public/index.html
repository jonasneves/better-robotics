Hard refresh
This will erase:
  • App cache and service worker (re-downloaded on reload)
  • Profile name, paired robots, settings
  • User scripts (the code in the Scripts editor)
  • Saved passwords, SSH keys, trust store
  • Pip replay history
  • Local Pip model weights (≈1.2 GB re-download if you had it installed)
  • Auth tokens (you'll need to sign in again)

Bluetooth pairing is controlled by the browser, not this app. If a robot won't reconnect after the refresh, revoke it at chrome://bluetooth-internals/#devices and re-pair.
